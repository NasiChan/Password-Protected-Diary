About My Diary

This project is a simple web diary designed to practice HTML, CSS, and JavaScript. It features password-protected entries and responsive design.

About the Creators

The project was built by Nastaran Pedari and Shreya Achari, who are both working on this project to improve their web development skills.

Nastaran Pedari: Focused on the behavior and functionality of the diary, ensuring that features such as error handling, entry editing, and secure login work smoothly. Nastaran is responsible for implementing JavaScript logic, such as displaying error messages when credentials are invalid, editing and deleting entries, and ensuring that the system logs out properly without losing data.

Shreya Achari: Worked on the structural side of the project, focusing on making the user interface intuitive and visually appealing. Shreya was responsible for the HTML and CSS layout of the diary, including the design and structure of the taskbar, where users can easily navigate to new and existing diary entries. Shreya implemented a clean, user-friendly design with a focus on responsiveness, ensuring that the diary works well across different screen sizes and devices.

Features of My Diary

  • Secure login system to protect your personal entries
  • Ability to view, edit, and delete previous entries
  • Error handling for invalid credentials
  • Personalized titles for entries for better organization
  • Option to search for specific entries quickly
  • Responsive design that adjusts to different screen sizes
